To install the AIRROHR firmware, please connect the ESP8266 board to the computer and then press the 'Install firmware' button. The firmware will then be installed on the ESP8266. As usual, the ESP8266 will then set up an access point with the IP address 192.168.4.1, i.e. a WLAN named 'airrohr- ...' should now be visible (search/display available WLANs on your smartphone/PC). Connect to this network, if a password is requested, enter the password 'airrohrcfg' and then enter the URL 192.168.4.1 in your web browser to access the configuration page.
