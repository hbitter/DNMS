DNMS Web Flasher pour AIRROHR ESP8266 Firmware

Pour installer le firmware AIRROHR, veuillez connecter la carte ESP8266 à l'ordinateur puis appuyer sur le bouton « Installer le firmware ». Le firmware est ensuite installé sur la carte ESP8266. Comme d'habitude, l'ESP8266 configure ensuite un point d'accès avec l'adresse IP 192.168.4.1, c'est-à-dire qu'un WLAN portant le nom « numéro airrohr » doit maintenant être visible (recherche/affichage des WLAN disponibles sur le smartphone/PC). Pour vous connecter à ce réseau, si un mot de passe est requis, saisissez le mot de passe 'airrohrcfg' puis saisissez l'URL 192.168.4.1 dans le navigateur web pour accéder à la page de configuration.
